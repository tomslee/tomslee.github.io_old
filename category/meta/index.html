Site setup

The site is set up with: GitHub pages, to host the published static content and serve it. Hugo, to generate the published static content from source. The Academic theme, even though I’m not an academic, to style the web site presentation. Org Mode with ox-hugo, to write the material. Although some files, including this, are written directly in markdown using Vim or Microsoft VSCode. I’m currently working on the content in an Ubuntu WSL2 instance on Windows, and using MobaXterm and/or Windows Terminal to get at it.
